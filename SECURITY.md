# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any Agent Mag repository, please report it responsibly.

**Do not open a public issue.**

Instead, email [hello@theagentmag.com](mailto:hello@theagentmag.com) with:

- A description of the vulnerability
- Steps to reproduce
- The potential impact

We will acknowledge your report within 48 hours and aim to release a fix within 7 days for critical issues.

## Supported Versions

We provide security updates for the latest major version of each tool. If you're running an older version, please upgrade first and check if the issue persists.

## Scope

This policy applies to all repositories under the [Agent-mag](https://github.com/Agent-mag) GitHub organization.
