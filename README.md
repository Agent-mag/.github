- **Organization profile** (`profile/README.md`) — the README shown on [github.com/Agent-mag](https://github.com/Agent-mag)

- **Code of conduct** — community standards
- **Security policy** — how to report vulnerabilities
- **Issue & PR templates** — consistent formats across repos
